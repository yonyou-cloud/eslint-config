# @yonyou-cloud/eslint-config

## Usage

```bash
npm i @yonyou-cloud/eslint-config -D
```

```js
// .eslintrc.js for js.
module.exports = {
  extends: ['@yonyou-cloud/eslint-config']
}

// .eslintrc.js for react.
module.exports = {
  extends: ['@yonyou-cloud/eslint-config/react']
}

// .eslintrc.js for vue.
module.exports = {
  extends: ['@yonyou-cloud/eslint-config/vue']
}
```

## Included

+ eslint:recommended
+ standard
+ eslint-plugin-vue
